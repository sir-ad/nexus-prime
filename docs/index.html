6 Tools. One Protocol.

Every tool is a native MCP endpoint. Call them the same way you call any tool — your agent framework does the rest.

💾 nexus_store_memory
Write a finding, insight, or decision to the Nexus Cortex. Priority 0.9+ auto-promotes to long-term SQLite. High-priority stores trigger Zettelkasten linking.
→ id: string (memory UUID)
🧠 nexus_recall_memory
Semantic nearest-neighbor search across all memory tiers. Bridges synonym gaps via 128-dim TF-IDF embeddings. Works even if exact words don't match.
→ string[] (top-k memory contents)
📊 nexus_memory_stats
Health check for the memory system. Shows item counts across all 3 tiers, top tags, Zettelkasten link count, and oldest entry timestamp.
→ { prefrontal, hippocampus, cortex, totalLinks, topTags }
⚡ nexus_optimize_tokens
Pre-flight reading plan before touching a file. Scores files by relevance to your task and assigns read, outline, or skip. Saves ~55% tokens on average.
→ ReadingPlan { files[], totalTokens, savings }
👻 nexus_ghost_pass
Read-only pre-flight analysis. Returns risk areas, a reading plan, and suggested worker approaches — without modifying anything. Like a dry run before a big task.
→ GhostReport { riskAreas, workerAssignments, readingPlan }
🛡️ nexus_mindkit_check
Machine-check an action against 6 guardrail rules before executing. Returns a score 0-100 with violations and actionable suggestions. Block before you break.
→ { passed, score, violations, warnings, summary }

// module: memory

Memory that
survives restarts.

Three-tier architecture modelled on human cognition. Everything persists to SQLite. Semantic recall bridges synonym gaps via TF-IDF embeddings.

Prefrontal
7
Active working set. Instant recall. The 7 most relevant items for the current session. Automatically populated by recall().
Hippocampus
200
Recent context window. Holds the last ~200 memories across recent sessions. Fast in-RAM access for current context.
Cortex
∞
Long-term permanent storage. SQLite-backed, survives all restarts. High-priority memories auto-fission here with Zettelkasten links.
HYBRID RECALL FORMULA
score = 0.50×vector_similarity + 0.25×priority + 0.15×recency + 0.10×access_count
Semantic TF-IDF 128-dim vectors bridge synonym gaps. "login broken" recalls "authentication bug" even with zero word overlap.

6 rules.
Machine-enforced.

Mindkit guardrails run before any significant operation. Block before you break. Warn before you overspend.

RULE ID TRIGGERS WHEN SEVERITY SUGGESTION
TOKEN_BUDGET context > 100k tokens 🚫 block Use nexus_optimize_tokens first
TOKEN_WARN context > 70k tokens ⚠️ warn Outline-only reads for large files
DESTRUCTIVE_GUARD isDestructive: true 🚫 block Request explicit user confirmation
BULK_FILE_GUARD modifying > 10 files ⚠️ warn Run nexus_ghost_pass first
NO_PROD_WRITES /etc, /usr, /bin paths 🚫 block Use phantom worktree instead
MEMORY_FIRST "research" / "look up" ℹ️ info Call nexus_recall_memory first
// getting started

Up in 4 steps.

Works with Claude Desktop, AntiGravity, Gemini, or any MCP-compatible agent framework.

1

Clone and build

Install dependencies and compile the TypeScript source.
git clone https://github.com/sir-ad/nexus-prime cd nexus-prime && npm install && npm run build
2

Wire as MCP server

Add to your agent's MCP config (Claude Desktop / AntiGravity settings).
{ "nexus-prime": { "command": "node", "args": ["/path/to/nexus-prime/dist/cli.js", "mcp"] }}
3

Add AGENTS.md to your project

Copy AGENTS.md to your project root so any agent knows the protocol automatically.
cp /path/to/nexus-prime/AGENTS.md ./AGENTS.md
4

Start building

Your agent now has 6 native MCP tools and persistent memory across all sessions.
→ nexus_recall_memory("today's task") → nexus_optimize_tokens("...") → nexus_mindkit_check("...")
