What it does

Five pillars of
agent intelligence.

Every AI agent starts each session with amnesia, burns tokens reading entire files, and works alone. Nexus Prime eliminates all three problems.

🧠

3-Tier Persistent Memory

Stop the "cold start" problem forever. Every insight, bug fix, and architectural decision is stored across sessions. Prefrontal (active working set) → Hippocampus (session cache) → Cortex (SQLite long-term knowledge with 820+ Zettelkasten links).

nexus_store_memory · nexus_recall_memory
⚡

Token Supremacy & CAS

HyperTune™ formulates file-reading as a Greedy Knapsack Problem. Paired with Continuous Attention Streams (CAS) and learned codebooks, it maximizes information gain within a strict token budget — saving 50-90% of context costs.

nexus_optimize_tokens · nexus_hypertune_max
🐝

Phantom Worker Swarms

Parallelize complex tasks using isolated Git Worktrees. Ghost Pass performs read-only risk analysis, while the Merge Oracle evaluates AST diffs using Byzantine consensus, Pearson correlation, and Hierarchical Synthesis.

nexus_ghost_pass · nexus_spawn_workers
🌌

Quantum Entanglement (Phase 9A)

Replaces explicit IPC messaging with a Shared Quantum-State Vector. Agents share state in a high-dimensional Hilbert space. When one agent acts, the state collapses via Born rule sampling, causing entangled agents to automatically make correlated decisions.

nexus_entangle · nexus_net_sync
🛡️

MindKit Guardrails

Machine-checked safety before every destructive operation. Scores actions 0-100 against safety protocols. Blocks `rm -rf`, token window blowouts, and untested mutations before they reach the shell via HITL checkpoints.

nexus_mindkit_check · nexus_audit_evolution

UX Interaction Layer

How agents
talk to each other.

Three communication channels enable real-time observability, cross-worker synchronization, and cross-machine knowledge sharing.

📡

Event Bus → Dashboard

Every engine emits strongly-typed events (42 types). The Event Bus broadcasts via SSE to the real-time dashboard — see memory stores, token savings, and phantom dispatches live.

nexusEventBus.emit('memory.store', {...})
🔗

POD Network

Phantom Workers exchange findings through a local file-backed pub/sub mesh. Workers publish observations with confidence scores, and the Merge Oracle synthesizes consensus autonomously.

podNetwork.publish(workerId, content)
🌐

NexusNet Federation

Share knowledge across machines using a GitHub Gist as a lightweight relay. Agents anonymously publish insights and sync learnings from other Nexus Prime instances — privacy-first with TTL expiration.

nexusNetRelay.publish('knowledge', {...})
Getting Started

Running in
30 seconds.

Nexus Prime integrates with any MCP-compatible agent: Claude, Gemini, GPT, or custom agents.

# Clone and build
git clone https://github.com/sir-ad/nexus-prime.git
cd nexus-prime
npm install && npm run build

# Add to your agent's MCP config
{
  "nexus-prime": {
    "command": "node",
    "args": ["/path/to/nexus-prime/dist/cli.js", "mcp"]
  }
}

# Verify — call from your agent
nexus_memory_stats() # → Returns cortex health
