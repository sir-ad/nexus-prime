# Nexus Prime v3.12.1

## Summary
`v3.12.1` is a recovery-and-operator patch release. It ships the current dashboard cockpit improvements, restores graph memory to the live recall path, seeds first-run knowledge fabric automatically, and makes persisted runtime activity survive dashboard restarts more cleanly.

## What Changed
- Added dashboard focus mode so core widgets can expand into a full workspace without losing state
- Tightened dashboard density with smaller rails, more efficient interior spacing, and clearer RAG injection status
- Wired graph memory into the live memory engine so graph stats and recall are active in production
- Auto-seeded a bounded project bootstrap RAG collection from repo context on first run when no collection is attached
- Persisted and rehydrated dashboard event history, with archive rotation for `events.jsonl`
- Seeded starter local hook and automation files into fresh workspace `.agent` scaffolds

## Dashboard Improvements
- Graph, memory snapshots, runtime events, operator actions, and major analysis panels now support maximize/restore focus mode
- Token surfaces refresh from runtime and knowledge activity so lifetime totals feel live while remaining persisted-runtime truth
- Knowledge mode now shows an explicit RAG injection path for `Attached`, `Retrieved`, `Selected`, `Used in Planner`, `Used in Packet`, and `Used in Runtime`
- Interior typography and spacing are tighter so more of the graph-first cockpit remains visible without sacrificing legibility

## Runtime and Reliability Improvements
- Graph memory is initialized beside the primary memory store and backfilled when empty
- Memory recall now blends graph matches with the main vector-backed memory engine
- First-run bootstrap/orchestrate attaches a bounded project context collection when the runtime has no RAG corpus yet
- Event history reloads from persisted JSONL logs after restart and rotates at 10 MB with three archives
- Fresh workspace bootstraps now expose visible starter hooks and automations instead of empty folders
