# Nexus Prime v3.2.0

## Title
Full Promise Closure

## Summary
This release promotes the remaining partial runtime surfaces into real product behavior. Nexus Prime now ships selectable advanced backends, bundled domain skill/workflow packs, derivation-capable runtime learning, real workflow control surfaces, and a live dashboard console backed by the same execution ledger used by CLI and MCP.

## Highlights
- Added bundled domain packs for marketing, product, backend, frontend, sales, finance, workflows, and orchestration.
- Added a workflow runtime with generation, deployment, execution, derivation, and promotion flows.
- Promoted backend selection into real execution behavior for temporal/hyperbolic memory, meta-compression, and experimental DSL compilation.
- Expanded the shared runtime into a multi-role graph with planner, coder, verifier, skill-maker, and research-shadow manifests.
- Rebuilt the dashboard into a runtime console with JSON APIs, SSE streaming, health panels, and dashboard smoke coverage.
- Fixed the GitHub Pages workflow syntax so docs deploy cleanly again.
