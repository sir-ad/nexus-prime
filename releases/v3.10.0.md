# Nexus Prime v3.10.0

## Summary
Nexus Prime now restores the intended external-client workflow: bootstrap first, orchestrate second. This release adds a dedicated `nexus_session_bootstrap` MCP entry point, defaults external clients to an `autonomous` tool profile, installs compact client-native bootstrap instructions during setup, and tightens the public docs so the README and website reflect the real orchestrator-first product.

## Highlights
- New `nexus_session_bootstrap` tool for session-start context, memory recall, stats, shortlist guidance, and token-optimization expectations
- Curated `autonomous` vs `full` MCP tool profiles, with bootstrap and orchestration prioritized for external clients
- `nexus-prime setup` now installs compact bootstrap instructions for Cursor, Windsurf, Claude Code, Opencode, and Antigravity/OpenClaw
- Runtime/dashboard snapshots now expose bootstrap/orchestrate compliance, tool-call chains, skip reasons, and client-instruction state
- Public docs refreshed around the current product surface, plus a new public-surface scan to catch stale claims and obvious leaks before release
