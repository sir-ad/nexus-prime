# Nexus Prime v3.1.0

## Title
Real Runtime Execution

## Summary
This release turns Nexus Prime's execution path into a real worktree-backed runtime. `execute()`, swarm orchestration, MCP runtime tools, and NXL execution now produce real diffs, verifier output, merge decisions, rollback behavior, and artifact directories instead of simulated completion text.

## Highlights
- Added a shared `SubAgentRuntime` for single-agent, swarm, MCP, and NXL execution.
- Added runtime skill artifacts with guarded hot deployment and rollout tracking.
- Added backend contracts for memory, compression, consensus, and DSL compilation so advanced implementations can evolve behind stable runtime interfaces.
- Updated CLI execution to accept real action files, verification commands, worker counts, and NXL input files.
- Hardened runtime diff capture, state storage, and worktree isolation for sandboxed/headless environments.
