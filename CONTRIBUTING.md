# Contributing to Nexus Prime

Thank you for your interest in contributing to Nexus Prime! Our mission is to build the ultimate open-source Cognitive OS for autonomous agent swarms.

## 🛠️ Development Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sir-ad/nexus-prime.git
   cd nexus-prime
   npm install
   ```

2. **Build the framework:**
   ```bash
   npm run build
   ```

3. **Verify tests pass:**
   ```bash
   npm run test
   ```

## 🧩 Pull Request Process

1. **Check the Issue Tracker**: State your intentions by finding an existing issue or opening a new one to discuss the proposed changes.
2. **Branch Naming**: Follow our naming conventions (`feature/add-foo`, `fix/resolve-bar`, `docs/update-baz`).
3. **Commit Messages**: Use Conventional Commits (`feat: ...`, `fix: ...`, `chore: ...`).
4. **Pass Checks**: Ensure `npm run lint` and `npm run test` pass perfectly before submitting. 
5. **Update Docs**: If you change an API or add a tool, update the `README.md` and `docs/index.html` accordingly.

We look forward to reviewing your PR and welcoming you to the Nexus Prime core community.
