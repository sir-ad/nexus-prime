# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| v1.5.x  | :white_check_mark: |
| < 1.5   | :x:                |

## Reporting a Vulnerability

Please report security vulnerabilities by creating a new **Security Advisory** on GitHub or emailing the maintainers directly (sir.ad@example.com). 

We take all security vulnerabilities seriously. Please **do not** open public issues for zero-day vulnerabilities or unpatched exploits. We will acknowledge receipt of your vulnerability report and strive to send you regular updates about our progress.
